Privacy Policy

We respect user privacy. This policy describes how Edutalk collects and uses data when users connect their TikTok account.

Data we process

  • OAuth authorization data (e.g., access token/refresh token) provided by TikTok after user consent.
  • Basic account identifiers returned by TikTok APIs (as permitted by granted scopes).
  • Posting metadata (caption, schedule time) provided by the user.

How we use data

  • To authenticate requests to TikTok APIs on behalf of the user.
  • To manage posting workflows initiated by the user.
  • To maintain system security and logging.

Data sharing

We do not sell personal data. We share data only as required to operate TikTok API integrations and comply with laws.

Security

We apply reasonable technical and organizational measures to protect stored credentials and data.
